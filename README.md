# Cybersecurity Career Roadmap with NotebookLM

An AI-assisted study project exploring the technical skills, certifications, and learning paths required to start a career in cybersecurity.

---

# Overview

This repository was developed as part of the DIO NotebookLM challenge, combining Artificial Intelligence, cybersecurity research, prompt engineering, and knowledge organization using NotebookLM and GitHub.

The main purpose of this project is to investigate:

> What technical skills and certifications are necessary to start a career in cybersecurity.

Using NotebookLM as an AI-powered research assistant, this project documents the complete learning process, including source curation, prompt experimentation, troubleshooting, AI-assisted summaries, and the creation of a structured cybersecurity study mini-guide.

Rather than simply generating AI answers, the project focuses on demonstrating:

- Active learning;
- Critical thinking;
- Prompt engineering;
- Research organization;
- And technical documentation practices.

---

# Project Objectives

The primary objectives of this project are:

- Identify the foundational technical skills required for cybersecurity beginners;
- Research beginner-friendly cybersecurity certifications;
- Understand realistic entry paths into cybersecurity without a university degree;
- Explore how Artificial Intelligence can support active learning workflows;
- Practice prompt engineering and iterative questioning;
- Document AI limitations, troubleshooting, and response refinement;
- Build a reusable cybersecurity study guide for future learning and revision.

# Prompt Engineering Process

A major focus of this project is documenting the interaction process with NotebookLM and demonstrating how prompt refinement improves AI-generated responses.

The project includes:
- exploratory prompts;
- iterative prompt refinement;
- troubleshooting;
- validation of AI-generated information;
- and documentation of limitations and insights.

## Example Prompts

### Foundational Skills

```text
What are the foundational technical skills required for cybersecurity beginners?
```

### Cybersecurity Fundamentals

```text
What are Cybersecurity?
```

### Career Entry Without a Degree

```text
How is possible to someone that doesn't have a degree enter the area? And what is a real expected time to get the first job?
```

### Beginner Certifications

```text
What are the best free certifications to get started?
```

# Lessons Learned During Prompt Testing

Throughout the project, several important observations were identified:

- Broad prompts tend to generate generic answers;
- More contextual prompts improve response quality significantly;
- AI-generated career advice requires critical evaluation;
- Reliable sources are essential for validating information;
- Prompt engineering is an iterative process.

The repository documents both successful and imperfect interactions to demonstrate the reasoning process behind the final results.
